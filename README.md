# Git Commit Mood Tracker

A Visual Studio Code extension that analyzes recent Git commit messages
and classifies them as **Positive**, **Neutral**, or **Negative** using
a Python-based sentiment analyzer.  
Results are displayed in a clean visual webview inside VS Code.

---

## Features

- Reads last 50 Git commit messages
- Performs mood/sentiment analysis using Python
- Displays a visual summary (bars + details)
- Works on any local Git repository

---

## Requirements

Make sure these are installed **before running**:

- **Git** (available in PATH)
- **Python 3.x**
- **Node.js (v16 or above)**
- **Visual Studio Code**

Check versions:
```bash
git --version
python --version
node --version
```
